  [Summeray]
    This tool can be used to filter project assets those name are not
    match the asset pattern in Unity editor.
--------------------------------------------------------------------------
  [Environment]
    This tool applies to Unity3D 5.0, .Net Framework 3.0 or above version.
--------------------------------------------------------------------------
  [Usage]
    Find the menu item "Tool/Asset Filter" in Unity editor's menu bar and
    click it or press combination key Alt+A to open the "Asset Filter"
    editor window.
    Click the "Browse" button to select the "Target Directory".
    Click the "New" button to create the "Pattern Settings", select it
    and config it in Unity Inspector.
    Click the "Check" button to check the target assets name.
    The filter assets will be display in the "Mismatch Assets" area, you
    can click a asset to focus it in the Unity Project.
--------------------------------------------------------------------------
  [AssetPattern]
    Use regular expressions to define the AssetPattern.
